# AITS Tirupati - Digital Transformation Project

This repository contains the Phase 1 deployment of the **Annamacharya Institute of Technology and Sciences (Tirupati)** official web portal.

## 🚀 Vision
To transition AITS from a standard web presence to an **AI-First Autonomous Campus** interface.

## 🛠 Tech Stack
- HTML5 / CSS3 (Inter Typography)
- JavaScript (ES6+)
- Mobile-Responsive Design

## 📈 Roadmap
- [ ] Integration of AI-Chatbot for Admissions.
- [ ] Real-time Student Placement API.
- [ ] Virtual 360 Campus Tour.
